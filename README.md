# AutoChronicle
Welcome to the description of the AutoChronicle! This application is a desktop application designed to facilitate access to the fascinating world of automobiles. It is nothing less than a true encyclopedia for automotive enthusiasts. It includes historical information about all known automotive companies, presented with illustrations and a brief summary. Enriched with an incredible design and the ability to change languages, this guide-like application is an invaluable companion for every beginner, and perhaps even a seasoned car enthusiast.
